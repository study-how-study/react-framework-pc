UdSwitch Antd Switch

扩展内容：

1. 可设置选中和不选中时的值，而不是固定的 true 或 false。
2. 在表单中使用时不用再单独设置 valuePropName。

## API

> 支持通过 `UdSwitch.defaultProps` 配置 `默认 props`。
