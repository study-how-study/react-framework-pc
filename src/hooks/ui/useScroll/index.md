主要是 元素的滚动条相关的信息，比如当前滚动条位置(后期根据项目需求扩展其他信息)。

参考的 [ahook useSize](https://ahooks.js.org/zh-CN/hooks/dom/use-scroll)

## API

`const Position = useScroll(target?: Target);`

## Params

| 参数   | 必传 | 说明 |                 类型 | 默认值   |
| :----- | ---- | :--: | -------------------: | :------- |
| target | -    | 容器 | HTMLElement/Document | document |
